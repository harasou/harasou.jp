hexo の 404 File not found さようなら

2015/6/28

Mac 上で hexo の記事を書いていると、記事へのリンクが 404 File not found になることがあった。 以前の記事では、sed でダサい対応をしていたのだが、@kazu69 が hexo-plugin を作成してくれました。

npm: hexo-filter-permalink-normalization

導入手順

plugin にしてくれたので、導入はホント簡単。blog を管理しているディレクトリで、npm install し、plugin の設定をするだけ。

  1. plugin インストール

    npm install hexo-filter-permalink-normalization --save

  2. plugin の設定

    hexo config permalink_normalization.type nfc

感想

自分はプログラミングがほとんどできないので、こんな plugin は作成できないが、困ってると話をすると @atani が sed ではなく、ちゃんと node で対応するやり方を見つけてくれたり、 @kazu69 が plugin を公開してくれたり。

OSS って、使うだけでも便利だけど、それに関わっていくと、さらに良さがわかる。
